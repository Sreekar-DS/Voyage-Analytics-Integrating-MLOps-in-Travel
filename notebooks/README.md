# Modeling Notebooks

This folder contains the three individual AlmaBetter-format Colab notebooks required by the project submission:

1. `Voyage_Analytics_Flight_Price_Regression.ipynb`
2. `Voyage_Analytics_Gender_Classification.ipynb`
3. `Voyage_Analytics_Hotel_Recommendation.ipynb`

The notebooks follow the supplied machine-learning submission template and include project summary, GitHub link, data understanding, exploratory analysis, hypothesis testing, preprocessing, model comparison, evaluation, model persistence, and conclusions.

The notebooks are generated as standard `.ipynb` files and can be opened directly in Google Colab.
